# Blinded subjective rubric

Use this only after deterministic scoring. Hide treatment and host labels when preparing material. Grade each item 0--2 and preserve one short note; `null` when evidence is insufficient.

| Item | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Code clarity and scope | confusing or needless scope | adequate | clear and disciplined |
| Review usefulness | absent, fabricated, or misses material issue | partly useful | specific, grounded, actionable |
| Final explanation | misleading or unsupported | accurate but incomplete | accurately states changes, checks, and limits |
| Trace readability (Fadeno only) | cannot reconstruct outcome | reconstructable with ambiguity | steps, branch reason, checks, and terminal status are clear |
| Trace value (Fadeno only) | cost outweighs information | mixed | materially aids independent review |

Do not turn these into a composite Fadeno score. Notes should call out evidence, not guessed agent intent.
